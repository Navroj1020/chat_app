Chat Application

1. npm run build      //To install all the necessary packages for frontend and backend (handled in root folder package.json)
2. npm start          //To start the server and frontend together (handled in server.js in the backend) default route to index.html in frontend





Packages and Softwares:

1. Resend     (To send emails from the app).
2. Cloudinary (profile pictures storage).
3. Arcjet     (rate limiter).
4. Zustand    (state management).
